# Elitras Swap Мод для Minecraft 1.21.1 (NeoForge)

Это мод, который автоматически переключает экипировку между элитрами и нагрудником во время боя на лету.

## ✨ Функции

- 🎯 **Автоматическое переключение на броню**: Когда вы берете булаву в руки, элитры автоматически меняются на нагрудник
- 🎆 **Автоматическое переключение на элитры**: Когда вы берете фейерверки в руки, нагрудник автоматически меняется на элитры
- 🚀 **Без необходимости нажимать прыжок**: После надевания элитр вы можете продолжать летать
- ⚔️ **PvP оптимизация**: Идеально для боев на лету
- 💾 **Автоматическое управление инвентарем**: Мод автоматически находит нужную броню или элитры в инвентаре

## 📦 Установка

### Способ 1: Скачать готовый мод
1. Скачайте последний файл мода `elitras-swap-1.0.0.jar` из [релизов](https://github.com/example/elitras-swap/releases)
2. Поместите его в папку `mods` вашего Minecraft клиента:
   - Windows: `%APPDATA%\.minecraft\mods\`
   - macOS: `~/Library/Application Support/minecraft/mods/`
   - Linux: `~/.minecraft/mods/`
3. Убедитесь, что у вас установлен **NeoForge 1.21.1**
4. Перезапустите Minecraft

### Способ 2: Собрать мод самостоятельно
1. Убедитесь, что у вас установлены:
   - Java 21 или выше
   - Git

2. Клонируйте репозиторий:
```bash
git clone https://github.com/example/elitras-swap.git
cd elitras-swap
```

3. На Windows запустите:
```bash
build.bat
```

На Linux/macOS запустите:
```bash
./gradlew build
```

4. Готовый мод будет в файле `build/libs/elitras-swap-1.0.0.jar`
5. Поместите его в папку `mods`

## 🎮 Как использовать

### Во время полета на элитрах:

1. **Для атаки булавой**:
   - Возьмите булаву в руку
   - Мод автоматически надевает нагрудник
   - Атакуйте врагов, защищенные броней

2. **Для использования феерверков**:
   - Возьмите фейерверки в руку
   - Мод автоматически надевает элитры
   - Продолжайте летать

### Требуемое оборудование:

Вам потребуется в инвентаре:
- ✅ Элитры (1 пара)
- ✅ Нагрудник (любой уровень)
- ✅ Булава (для атак)
- ✅ Фейерверки (для контроля полета)

## 🛠️ Технические детали

### Как это работает

Мод слушает события изменения предметов в руках игрока:

```
Берет булаву → Находит броню в инвентаре → Надевает ее
                ↓
            Элитры отправляются в инвентарь
                ↓
            Готовы к использованию

Берет фейерверки → Находит элитры в инвентаре → Надевает их
                    ↓
                Броня отправляется в инвентарь
                    ↓
                Готовы к полету
```

### Поддерживаемые типы брони

- ⚒️ Алмазный нагрудник
- ⚫ Нетеритовый нагрудник
- 🔨 Железный нагрудник
- 🥇 Золотой нагрудник
- 🧥 Кожаный нагрудник

## 🐛 Решение проблем

### Мод не работает после установки
- Убедитесь, что NeoForge 1.21.1 установлен
- Проверьте, что мод находится в папке `mods`
- Перезапустите клиент

### Броня не переключается
- Убедитесь, что в инвентаре есть оба предмета (броня и элитры)
- Проверьте, что булава и фейерверки находятся в главной руке (не в левой)
- Попробуйте обновить мод до последней версии

### Не могу летать после взятия элитр
- Нажмите Space (прыжок) один раз, затем еще раз для полета
- Или используйте фейерверки для лучшего управления полетом
